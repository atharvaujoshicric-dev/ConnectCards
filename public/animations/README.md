# Character animations

Drop `.lottie` files here and reference them by filename in
`<CharacterAnimation src="filename.lottie" ariaLabel="..." />`
(see `components/marketing/character-animation.tsx`).

Suggested filenames once you have assets, to match where they're
intended to slot in:

- `tap-moment.lottie` — a character tapping a card against a phone
- `paper-to-recycle.lottie` — a character replacing paper cards with one metal card

Sourcing options (see conversation / project docs for full instructions
and prompts):
1. LottieFiles marketplace (free, fastest): lottiefiles.com
2. LottieFiles AI generator (paid Creator plan)
3. Commission via Fiverr/Upwork ("Lottie animation illustrator")

This folder is empty until you add files — nothing references a
character animation yet, so the app builds and runs fine without them.
